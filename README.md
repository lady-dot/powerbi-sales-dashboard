# powerbi-sales-dashboard
Interactive Sales dashboard built in Power BI with multiple charts and user interface

## Overview
This project is an interactive sales dashboard built using Power BI. The dashboard visualizes sales data, provides key business insights, and allows users to explore metrics interactively.

## Dataset
The dataset was obtained from [source link / YouTube tutorial description]. Data has been cleaned and preprocessed for analysis.

## Tools & Technologies
- Power BI Desktop
- Data cleaning & transformation in Power Query
- Interactive charts, slicers, and filters

## Methodology
1. Data cleaning and preprocessing
2. Building relationships between tables
3. Creating charts and visuals
4. Customizing layout, colors, and formatting
5. Interactive dashboard deployment in Power BI Desktop

## Features
- Multiple interactive charts
- Filtering and drill-down capability
- Custom visual formatting

## Future Improvements
- Add predictive analytics or trend forecasting
- Deploy dashboard online via Power BI Service
